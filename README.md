# ArcGIS FLY

A "flight simulator" that allows you to control an ArcGIS SceneView from your mobile device through websockets.

Meant to be a modular app built on:
- ES2015
- Express
- Gulp
- Socket.io
- ArcGIS JS API (Dojo)

# Live at:
 - https://pure-dawn-16123.herokuapp.com/
 - http://bit.do/arcgisfly

#### To run:

- ```npm install```
- ```bower install```
- ```gulp```

#### Requirements:
gulp, bower, node
